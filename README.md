# AI Unit Test Generator

A lightweight Python tool that uses large language models to automatically generate unit tests from source code.

This project demonstrates how AI-assisted development tools can accelerate the software development lifecycle (SDLC) by generating test cases automatically.

Features:
- Generates pytest unit tests
- Includes edge cases and failure cases
- Supports human-in-the-loop validation

Technologies:
- Python
- LLM APIs
- Prompt engineering
